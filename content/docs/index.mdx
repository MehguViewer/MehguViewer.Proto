# MehguViewer.Proto

Welcome to the official documentation for the **MehguViewer** ecosystem. This site serves as the "Source of Truth" for all schemas, API specifications, and architectural decisions.

## Documentation Structure

We have reorganized the documentation to better reflect the "Three Pillars" architecture.

### [1. Getting Started](/docs/00-intro/01-getting-started)
Start here to understand the high-level architecture, the separation between Auth and Core, and the handshake workflow.

### [2. Authentication Flow](/docs/01-auth-flow)
Deep dive into the OIDC-compliant identity system. Learn how tokens are issued, verified, and how global blocklists work.

### [3. Protocol Specifications](/docs/02-protocol)
Understand the data models for Manga, Anime, and Novels. Learn about the polymorphic `Series` object and how federation works.

### [4. API Reference](/docs/03-api-reference)
Interactive documentation for the endpoints.
- **Auth API:** Identity & Profiles.
- **Core API:** Content & Social.
