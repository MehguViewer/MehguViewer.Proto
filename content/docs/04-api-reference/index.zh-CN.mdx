# API 参考

MehguViewer 协议分为两个不同的 API。您可以在下面浏览每个 API 的交互式文档。

## [Auth API](/api-reference/auth)
**身份提供者** API。
- **基础 URL：** `https://auth.mehgu.example.com`
- **关键端点：**
    - `/.well-known/openid-configuration`：发现。
    - `/api/v1/me`：用户配置文件。
    - `/api/v1/me/relationships`：屏蔽列表管理。

[打开交互式 Auth 文档 ->](/api-reference/auth)

## [Core API](/api-reference/core)
**内容交付** API。
- **基础 URL：** `https://core.mehgu.example.com`
- **关键端点：**
    - `/api/v1/series`：列出漫画/动画。
    - `/api/v1/series/{id}/units`：获取章节/剧集。
    - `/api/v1/comments`：社交互动。

[打开交互式 Core 文档 ->](/api-reference/core)
