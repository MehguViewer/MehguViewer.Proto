# API Reference

The MehguViewer Protocol is split into two distinct APIs. You can explore the interactive documentation for each below.

## [Auth API](/api-reference/auth)
The **Identity Provider** API.
- **Base URL:** `https://auth.mehgu.example.com`
- **Key Endpoints:**
    - `/.well-known/openid-configuration`: Discovery.
    - `/api/v1/me`: User Profile.
    - `/api/v1/me/relationships`: Blocklist management.

[Open Interactive Auth Docs ->](/api-reference/auth)

## [Core API](/api-reference/core)
The **Content Delivery** API.
- **Base URL:** `https://core.mehgu.example.com`
- **Key Endpoints:**
    - `/api/v1/series`: List Manga/Anime.
    - `/api/v1/series/{id}/units`: Get Chapters/Episodes.
    - `/api/v1/comments`: Social interactions.

[Open Interactive Core Docs ->](/api-reference/core)
